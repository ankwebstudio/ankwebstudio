About the Project

A personal Shopify project developed to explore professional e-commerce website development.

AamizNest is a fully responsive Shopify dropshipping store built for the Home & Kitchen niche. The project focused on creating a clean shopping experience, attractive product pages, responsive layouts, collections, and intuitive navigation.

This project allowed me to gain hands-on experience in Shopify theme customization, product management, branding, customer-focused UI design, and e-commerce best practices.

While the live store has been archived, it remains an important portfolio project demonstrating my Shopify development skills.
